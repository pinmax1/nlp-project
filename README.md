# Тема проекта
RAG-ассистент для подбора игр в Steam по запросу пользователя

# Описание проекта
Наш ассистент по запросу пользователя(который может содержать описание его любимых игр, предпочтения в жанрах, желаемый спектр оценок за игру) сможет подобрать ему список самых подходящих игр, подходящих под его критерии.
RAG в нашем случае нужен для получения списка игр, описаний игр, их оценок.

# Как это работает
Мы в промптах описываем LLM'ке тулзы, которые она может использовать в получении информации. 
LLM'ка должна эти тулзы вызывать в нужные моменты, когда ей не хватает информации.
Потом LLM'ка дожидается ответа тулзов, получает их обратно и собирает свой ответ на основе предыдущего контекста и полученного ответа.

То есть наша LLM'ка должна уметь запускаться в несколько раундов и вызывать тулзы
